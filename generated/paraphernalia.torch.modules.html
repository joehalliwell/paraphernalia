class WeightedSum(**components)[source]¶

Bases: torch.nn.modules.module.Module

More or less a weighted sum of named module outputs, but with special handling for negative weights.

In order for the weighting to make sense, the components needs outputs with the same shape and meaning. For loss functions outputs should be in the range [0,1]
